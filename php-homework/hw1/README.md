

    Создайте свой первый сайт на хостинге, придумав доменное имя для него.
    Используя данные для FTP-доступа, соединитесь по FTP, загрузите в директорию public_html предварительно созданный файл с именем index.html (в файл можно поместить любой HTML-размеченный контент).
    Проверьте доступ к сайту из браузера.
    Загрузите на сервер дополнительные файлы разных типов: картинки, документы, пр.
    Проверьте доступ к этим файлам, используя панель разработчика, изучите HTTP-заголовки запроса и ответа.
    Используйте панель разработчика, чтобы изучить заголовочную информацию о других сайтах в сети интернет.
    Составьте отчет о HTTP-заголовках: какими они бывают в запросах и ответах, какие они принимают значения (в общем и частном).
